TECHNOLOGY

Hardware

I have always enjoyed hardware. I've built a desktop computer, carried out some rudimentary laptop upgrades and repairs, built a RetroPie with a Raspberry Pi, and a large range of analog and digital effect pedals.

I find the fact that hardware is tangible quite appealing. As fun and interesting as it is to make intangilbe things, it is satisfying to have a physical object which I designed and built. This is especially true with guitar effects--someday, I would like to build my own amplifier and guitar as well.

Software

Examine my code on Github.

I initially became interested in software through video games. The prospect of creating a world fascinated me. While this interest still exists, (and I have some familiarity with a few game engines; Unity, Godot, and Love), I have not produced any original games worth mentioning (yet). In my quest to learn about making games, aside from sound-design and pixel art, I learned a good deal about various programming languages, including Lua, Python, JavaScript, CSS, HTML, Java, C, C++ and C#.

My work with effect pedals also lead me to working with assembly, particularly the SpinASM language for the Spin Semiconductor FV-1.

Most websites I have built from scratch have been projects done for the sake of learning. That includes this portfolio website. Additionally, my guitar pedal website mentioned above was built with Squarespace, and my literary blog mentioned below is built with Wordpress, so I am no stranger to that sort of technology as well.
